OpenAI 是一家颇具影响力的人工智能公司，三个月前罢免了备受瞩目的首席执行官，然后又重新上任，现在面临着新的戏剧性事件：来自世界上最富有的人之一、人工智能实验室的联合创始人埃隆·马斯克 (Elon Musk) 的诉讼。

马斯克起诉 OpenAI 及其首席执行官萨姆·奥特曼 (Sam Altman)，指控他们违反合同，将开发人工智能的利润和商业利益置于公共利益之上。 马斯克表示，OpenAI 与微软建立了数十亿美元的合作伙伴关系，这代表着他们放弃了精心开发人工智能的创始承诺。 并将该技术公开。

周四向旧金山高等法院提起的诉讼称，“OpenAI 已转变为最大科技公司微软的事实上的闭源子公司。”

这份长达 35 页的诉讼是这对前商业伙伴之间多年来一直在酝酿的争斗的最新篇章，它聚焦于人工智能领域尚未解决的问题：人工智能会改善世界还是毁灭世界，应该严格控制还是释放？

特斯拉首席执行官马斯克和奥特曼以及世界上任何人都帮助促成了这场辩论。 作为对当时谷歌正在做的人工智能的回应，马斯克先生于 2015 年帮助创立了 OpenAI。 马斯克认为，谷歌及其联合创始人拉里·佩奇对人工智能对人类造成的风险不屑一顾。

马斯克先生在 2018 年的权力斗争中离开了 OpenAI 董事会。该公司后来成为生成人工智能领域的领导者。 并创建了 ChatGPT，这是一个聊天机器人，可以生成文本并以类似人类的散文形式响应查询。 马斯克先生去年创立了自己的人工智能：一家名为 xAI 的公司，并表示 OpenAI 对该技术的风险关注不够。

这起诉讼也是一家陷入争议的公司的最新转折。 去年 11 月，OpenAI 董事会驱逐了奥特曼先生，并表示不再信任他来管理公司。 仅仅五天后，由于员工叛乱威胁到了公司的未来，他就被复职了。

硅谷内部人士认为，ChatGPT 背后的技术生成式人工智能是一项千载难逢的技术，它可以像 30 多年前的网络浏览器一样彻底改变科技行业。

“加州法院必须决定 OpenAI 在偏离其最初使命后必须做什么，”人工智能企业家、纽约大学心理学和神经科学名誉教授加里·马库斯 (Gary Marcus) 说。 “公众舆论法庭必须决定对马斯克的看法，马斯克对 OpenAI 的看法很合理，但也有自己的商业人工智能的兴趣和选择。”

OpenAI 拒绝就该诉讼发表评论。 《纽约时报》看到了周五下午发送给 OpenAI 员工的消息，奥尔特曼表示，他对马斯克关于构建人工智能的论点感到困惑。 为了人类的利益与建立企业是不一致的。

OpenAI 首席战略官 Jason Kwon 在《泰晤士报》看到的另一条消息中告诉 OpenAI 员工，公司领导层“坚决不同意”该诉讼。 他写道，马斯克的说法“并不反映我们工作或使命的现实”。

这起诉讼加剧了 OpenAI 面临的一系列问题。 该公司与微软的关系也面临美国、欧盟和英国监管机构的审查。 它因窃取受版权保护的材料来训练其聊天机器人而被《纽约时报》、多家数字媒体、作家和计算机程序员起诉。 美国证券交易委员会正在调查奥特曼先生和 OpenAI。

马斯克在诉讼中称，他之所以参与 OpenAI，是因为该公司是一家非营利组织，旨在为了“人类的利益”开发人工智能。 该诉讼称，其中的一个关键组成部分是使其技术开源，这意味着它将与世界共享底层软件代码。 相反，该公司创建了一个营利性业务部门，并限制对其技术的使用。

该诉讼寻求陪审团审判，指控 OpenAI 和 奥特曼先生违反合同、违反信托义务以及不公平的商业行为。 马斯克要求 OpenAI 向其他人开放其技术，并要求奥特曼和其他人偿还马斯克向该组织提供的资金。 OpenAI 总裁格雷格·布罗克曼 (Greg Brockman) 也被列为被告。

马斯克的论点取决于 OpenAI 和微软之间的密切合作关系。 2019 年，奥特曼先生谈判达成一项协议，微软同意向 OpenAI 投资 10 亿美元。 这家初创公司表示，它将专门使用微软的云计算服务来构建和部署其人工智能。 此后的几年里，微软向这家初创公司追加投资了 120 亿美元，并且成为 OpenAI 之外唯一一家获得使用 GPT-4 背后最强大的人工智能原始技术许可的公司。

谷歌、Meta 和法国初创公司 Mistral 等其他公司正在与其他公司和研究人员自由分享他们的一些最新技术。

该诉讼可能会让 OpenAI 面临漫长且具有侵入性的法律审查，从而更多地揭示奥特曼先生被解雇以及 OpenAI 从非营利组织转向营利性公司的情况。 这一变化由奥特曼先生于 2018 年底和 2019 年初发起，多年来一直是 OpenAI 背后中伤的根源，并导致董事会决定解雇他的首席执行官职务。

尽管马斯克多次批评 OpenAI 成为一家营利性公司，但他在 2017 年制定了一项计划，从奥特曼和其他创始人手中夺取人工智能实验室的控制权。 知情人士表示，马斯克计划将其转变为商业运营，将与他的其他公司（包括电动汽车制造商特斯拉）合作，并利用他们日益强大的超级计算机。 知情人士称，当他控制权的尝试失败后，他离开了 OpenAI 董事会。

去年，马斯克在《纽约时报》的 DealBook 峰会上发表讲话时表示，他想更多地了解 OpenAI 去年十一月发生的混乱情况，包括联合创始人伊利亚·苏茨克韦尔 (Ilya Sutskever) 为何联合其他董事会成员解雇奥特曼。 他表示，他担心 OpenAI 发现了人工智能的一些危险因素，作为诉讼的一部分，他的法律团队可以调查这个问题。

“我对 Sam 有复杂的感情，”马斯克在 DealBook 会议上说道。 他提到了《指环王》中的强大戒指，他补充道，“权力戒指可以腐败，而他拥有权力戒指。”

马斯克没有回应置评请求。

马斯克和奥特曼之间的争吵长期以来一直是硅谷的热门话题。 两人第一次见面是在参观马斯克的火箭公司 SpaceX 时，后来因为共同担心人工智能对人类构成的威胁而走到了一起。

根据该诉讼，OpenAI 的非营利地位是试图从新的人工智能中赚钱的公司高管和马斯克先生产生摩擦的一个主要来源，后者希望它仍然是一个研究实验室。

根据诉状，马斯克曾一度表示：“要么自己做点什么，要么继续将 OpenAI 作为非营利组织。” “在你们坚定地承诺留下来之前，我将不再为 OpenAI 提供资金，否则我就是个傻瓜，本质上是在为初创公司提供免费资金。 讨论结束了。”

该诉讼试图表明马斯克先生是 OpenAI 发展中不可或缺的人物。 诉讼称，从 2016 年到 2020 年，马斯克向 OpenAI 捐赠了超过 4400 万美元。 他还租用了公司最初在旧金山的办公空间并支付了每月的费用。 起诉书称，他亲自参与了对谷歌顶级研究科学家 Sutskever 先生的招聘，Sutskever 目前担任 OpenAI 的首席科学家。

诉讼称：“如果没有马斯克先生的参与以及大量的支持努力和资源，OpenAI 很可能永远不会起步。”

波士顿学院法学教授布莱恩·奎恩 (Brian Quinn) 表示，马斯克的投诉有力地证明了 OpenAI 已经放弃了自己的根基。 但是他说，马斯克先生可能没有资格提起诉讼，因为非营利组织法律将此类挑战限制为由非营利组织的会费成员、其自己的董事或特拉华州（OpenAI 注册地）的州监管机构提出。
